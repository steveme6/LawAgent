本项目提供了一个用于法律的智能体，可以提供知识问答，采用faiss向量检索以及langchain。
# 快速使用

## 使用uv（推荐）

1. 安装uv
```bash
curl -LsSf https://astral.sh/uv/install.sh | sh
```

2. 运行程序
```bash
uv run main.py
```

## 使用python

1. 安装python3.12
2. 安装依赖
```bash
pip install -r requirements.txt
```

3. 运行程序
```bash
python main.py
```
